1 数学定理环境

LaTeX 中提供了 amsthm 宏包设置数学定理环境，我们可以通过该宏包很好地定义定理环境。

1.1 设置定理环境

下面我们定义一个 Theorem ： 

\usepackage{amsthm}
\newtheorem{theo}{Theorem}
\begin{document}
\section{Theorem}
\begin{theo}
  This i s a Theorem.
\end{theo}
\begin{theo}
  This is a Theorem.
\end{theo}
\end{document}

1.2 定理序号

上面的例子中，定理是自动编号的，我们可以设置，让编号使用 section 号：

1.2.1 使用章节号 

\newtheorem{theo}{Theorem}[section]

当然，你可以设置成诸如 chapter, subsection 等，这些都是有效的。

1.2.2 去除编号

如果不需要编号，你可以在 newtheorem 后面加上一个星号(*) ，比如这样设置： 

\newtheorem*{nt}{Note}

1.2.3 使用相同编号

在我们定义了不同的定理环境后，这些环境默认是各自编号的，如果我们想让这些编号使用统一的计数，那么，在 \newtheorem 后面添加选项完成这样的设置： 

\newtheorem{defi}[theo]{Definition}

这样， defi 环境就继承了 theo 的属性，使得它们使用相同的编号计数：

共用编号

1.3 对定理说明

有时我们需要对定理进行说明，比如是什么什么定理，这时直接在 \begin{theo}后面添加就可以了： 

\begin{theo}{xxx Theorem}
A B C D
\end{theo}

1.4 定理样式

我们可以通过 \theoremstyle{} 来设置定理的样式： 

   \theoremstyle{theorem}
   \newtheorem{theo}{Theorem}[section]
   
   \theoremstyle{definition}
   \newtheorem{defi}[theo]{Definition}
   
   \theoremstyle{remark}
   \newtheorem*{nt}{Note}
   
   \begin{document}
   \section{Theorem}
   \begin{theo}
   This is a Theorem.
   \end{theo}
   \begin{defi}
   This is a Definition.
   \end{defi}
   \begin{nt}
   This is a Note.
   \end{nt}
