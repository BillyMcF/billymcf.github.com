9.1. 准备

TeX 对 .eps 格式的图片支持最好，所以，我们有必要将图片转换为 eps 格式。

下面是我正在使用的方法：

  • 使用 convert 命令

    $ convert 1.jpg 1.eps

  • 使用 sam2p

    我以前通过脚本使用 scrot 截图并直接转换为 eps 图片，下面是脚本内容。你可以把这个脚本放在 /usr/bin 下，然后赋予可执行权限: chmod +x seps，这样方便在其他地方使用。

    	#!/bin/bash
	if [ $# -ne 0 ]; then
	name=$1
	else
	name=tmp
	fi
	scrot -s $name.jpg
	sam2p $name.jpg $name.eps
	rm $name.jpg

    这样，就可以通过 $ seps 文件名使用这个脚本了。

9.2. graphicx 宏包

通过以下命令插入图片：

    \includegraphics[选项]{eps图片}
表 9.1. 插图选项
选项 说明
height 图形的高度（可为任何 TeX 度量单位)
totalheight 图形的全部高度（可为任何 TeX 度量单位）
width 图形的宽度 （可为任何 TeX 度量单位）
scale 图形缩放因子，比如 scale=0.5，即为原大小的一半
angle 旋转角度，以度为单位，顺时针为正
origin 指定图形绕某个点旋转
bb 设定 BoundingBox 的值

例 9.1. 插图选项顺序先后可能造成差别 

		\includegraphics[angle=90,totalheight=2cm]{tex.eps}
		\includegraphics[totalheight=2cm,angle=90]{tex.eps}
注意

第一条命令是 先旋转，后缩放。

第二条命令是 先缩放，后旋转。

所以，它们得到的结果是不一样的。

9.3. 图形标题

可以通过 \caption 命令设定图形的标题，它的位置决定标题是位于图形的上方还是下方：

例 9.2. 使用 \caption 命令设定图形标题 

		\begin{figure}
		\centering
		\caption{Caption}  %标题在上方
		\includegraphics[scale=.5]{tex.eps}
		\end{figure}
		
		\begin{figure}
		\centering
		\includegraphics[scale=.5]{tex.eps}
		\caption{Caption}  %标题在下方
		\end{figure}

9.4. 调整间距

标题的上下间距 \abovecaptionskip 和 \belowcaptionskip 缺省分别为 10pt 和 0pt,可以使用 \setlength 和 \addtolength 来修改它们。

例 9.3. 调整标题与图形间距 

		%标题在上
		\begin{figure}
		\setlength{\abovecaptionskip}{5pt}
		\setlength{\belowcaptionskip}{10pt}
		\centering
		\caption{Caption}
		\includegraphics[scale=.5]{tex.eps}
		\end{figure}
		
		%标题在下
		\begin{figure}
		\setlength{\abovecaptionskip}{0pt}
		\setlength{\belowcaptionskip}{10pt}
		\centering
		\includegraphics[scale=.5]{tex.eps}
		\caption{Caption}
		\end{figure}

提示

可以将 \setlength 放在导言区 [4]，这样可以对整个文档起作用。

也可以这样定义，使得标题在下方时使用 \bottomcaption，在上方时使用 \topcaption：

    \newcommand{\topcaption}{%
    \setlength{\abovecaptionskip}{0pt}%
    \setlength{\belowcaptionskip}{10pt}%
    \caption}

    \newcommand{\bottomcaption}{%
    \setlength{\abovecaptionskip}{10pt}%
    \setlength{\belowcaptionskip}{0pt}%
    \caption}#+end_src

9.5. 修改 Figure 字样

可以使用 \renewcommand 命令将 Figure 字样修改为 “图”或者其他字样：

	      \begin{figure}
	      \centering
	      \includegraphics[scale=.5]{tex.eps}
	      \renewcommand{\figurename}{图 }
	      \caption{Caption}
	      \end{figure}

[4] 导言区即为 \begin{document} 前的部分，通常用于设定文档相关选项。
