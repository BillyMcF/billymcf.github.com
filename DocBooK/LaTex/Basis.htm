1.1. 安装

建议安装 texlive 套装，里面包含有 LaTeX, xeTeX 等，很方便。

  1. 安装 Texlive

    # pacman -S texlive-bin texlive-cn texlive-latexextra latex-science

  2. 安装 CJK

    # pacman -S texlive-langcjk

  3. 安装中文字体

    我现在使用 xelatex 命令编译 tex 文件，可以直接使用系统中的字体。

    可以安装 ttf-ms-fonts[1]:

    # yaourt -S ttf-ms-fonts

    也可以使用 mkfontdir 和 mkfontscale 安装网上下载的字体。

1.2. Hello LaTeX

安装完成之后，我们可以编写一个简单的 tex 文件来测试一下：

      \documentclass{article}
      \begin{document}
      Hello,\LaTeX
      \end{document}

将这个文件保存为 test.tex，然后使用 latex[2] 命令进行编译

    $ latex test.tex

如果没有报错，会在同目录下生成 test.dvi 文件，可以使用 xdvi 等软件打开查看。当然，可以使用下面的命令使之生成 pdf 文件

    $ dvipdf test.dvi

或者可以使用 pdflatex 命令直接生成 pdf 文件

    $ pdflatex test.tex

1.3. 你好 LaTeX

下面我们测试一下中文支持是否正常

    \documentclass{article}
    \usepackage{CJKutf8}
    \begin{document}
    \begin{CJK*}{UTF8}{gbsn}
    你好，\LaTeX
    \end{CJK*}
    \end{document}

同样地，我们使用 pdflatex 命令来编译并查看生成的pdf文件。


[1] 这些字体即windows下的宋体、黑体、仿宋、隶书等，使用 ctex 宏包很方便。

[2] 当然可以使用诸如 pdflatex, xelatex等命令进行编译，直接生成pdf文件。
