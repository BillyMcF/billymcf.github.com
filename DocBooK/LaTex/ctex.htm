2.1. Linux下ctex宏包字体
CTEX 是中文 TeX (chinese TeX)的缩写。其宗旨是为广大中文TeX用户提供帮助。在使用 LaTeX 写中文文档的时候，ctex 宏包提供了十分便利的设置。

2.1. Linux下ctex宏包字体

安装 texlive 和 中文字体 [3]后，可以使用 xelatex 命令编译 tex 文件，但是会提示找不到 SIMKAI.TTF。使用 fc-list :lang=zh 可以看到，楷体和仿宋体后面都多出了 _GB2312。

这时候，需要修改 /usr/share/texmf-dist/tex/latex/ctex/fontset/ctex-xecjk-winfonts.def 文件。

改为如下：

    % ctex-xecjk-winfonts.def: Windows 的 xeCJK 字体设置，默认为六种中易字体
    % vim:ft=tex

    \setCJKmainfont[BoldFont={SimHei},ItalicFont={KaiTi_GB2312}]
    {SimSun}
    \setCJKsansfont{SimHei}
    \setCJKmonofont{FangSong_GB2312}

    \setCJKfamilyfont{zhsong}{SimSun}
    \setCJKfamilyfont{zhhei}{SimHei}
    \setCJKfamilyfont{zhkai}{KaiTi_GB2312}
    \setCJKfamilyfont{zhfs}{FangSong_GB2312}
    \setCJKfamilyfont{zhli}{LiSu}
    \setCJKfamilyfont{zhyou}{YouYuan}

    \newcommand*{\songti}{\CJKfamily{zhsong}} % 宋体
    \newcommand*{\heiti}{\CJKfamily{zhhei}}   % 黑体
    \newcommand*{\kaishu}{\CJKfamily{zhkai}}  % 楷书
    \newcommand*{\fangsong}{\CJKfamily{zhfs}} % 仿宋
    \newcommand*{\lishu}{\CJKfamily{zhli}}    % 隶书
    \newcommand*{\youyuan}{\CJKfamily{zhyou}} % 幼圆

    \endinput

或者你可以在你自己的 tex 文件中进行设置。


[3] 比如ArchLinux下安装ttf-ms-fonts,包含有宋体、仿宋，黑体，隶书等六种常用字体。当然，你可以拷贝或者上网下载这六种字体，然后使用 mkfontdir 和 mkfontscale 进行安装。
