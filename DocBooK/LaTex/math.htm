8.1. 数学环境

数学宏包是 amsmath

LaTeX 中，提供了以下几种数学环境：

  • $...$ 或者 \(...\)

    行内公式

  • \[...]\ 或者 $$...$$

    公式单独占一行

  • equation 环境

    公式独占一行，并且自动被添加编号

    如果不想公式被编号，则可以这样使用\begin{equation*}...\end{equation*}

如果你想像我一样使用 preview 查看数学公式，请参考这里preview 预览公式。如果提示找不到images，也许是因为你编译emacs及auctex时缺少了点参数。具体请参照auctex wiki。

8.2. 字符

普通字符在数学公式中含义一样，但需要使用 # $ % ^ & ~ _ \ { } 这些字符时，需要在他们前面加上\ , 例如 \{ 。

8.4. 上标和下标

用 ^ 表示上标，用 _ 表示下标。

    $$F(t) = \sum_{n=-\infty}^\infty F_n e^{j2 \pi nf_0 t}$$
重要
如果上标或者下标不止一个字符，你需要使用 {} 将它们包括在一起。

8.5. 分数、开方

  • 开方: \sqrt{}

  • n 次方: \sqrt[n]{}

8.7. 插入文本

有时我们需要在公式中输入添加文本，这时可以使用 \mbox{}: 

    $$E_T = \int_{-\infty}^\infty f_T^2(t)d_t = \int_{-\infty}^\infty |F_T(f)|^2d_f \ \ \mbox{(根据帕色瓦尔定理)}$$
