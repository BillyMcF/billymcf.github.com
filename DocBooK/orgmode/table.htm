3.1. 表格

org-mode 里制作表格非常的方便和有趣，还可以根据你设定的公式自动计算诸如total等，还可以自动编号。

3.2. 建立表格

建立表格非常简单，表格的列以 | 隔开，按 TAB 跳到下一单元格， S-TAB 跳到上一单元格。新行 |- 然后 C-TAB 插入水平线，建立下面： 

| 操作                  | 说明             |
|-----------------------+------------------|
| M-S-<right>           | 插入一列         |
| M-S-<left>            | 删除一列         |
| M-<right>/<left>      | 左右两列调换位置 |
| 行首插入 - 然后 C-TAB | 插入水平线       |
说明

这里只是显示的不同，在 org-mode 里面编辑表格是会自动对齐的，其中的字体需要设置一下，比如我设置的是 (set-default-font "WenQuanYi Zen Hei Mono-15") ，诸如 13 15 18 等字体大小都可以。如下图

3.3. 表格操作的一些常用快捷键

表 3.2. 表格操作快捷键

<TAB>跳到下一项
S-<TAB>跳到上一项
<RET>跳到同列下一行
S-<RET>如果是数字将向下自动编号，如果是其他，将复制
行首插入 - 然后 C-TAB插入水平线
C-c -插入水平线
C-c RET插入水平线并跳到其后
M-<left>与左列调换
M-<right>与右列调换
M-S-<left>删除一列
M-S-<right>增加一列
M-S-<up>/<down>删除/增加一行

当然还有很多的快捷键，可以参考 org-mode 主页。

C-x C-f打开文件
C-x C-v打开一个文件，取代当前缓冲区
C-x C-s保存文件
C-x C-w存为新文件
C-x i插入文件
C-x C-q切换为只读或者读写模式
C-x C-c退出Emacs

例 3.1. 从网页上复制表格到 org-mode

  1. 从网页上面复制，粘贴[5]到 Emacs 里面;

  2. 选中粘贴后的内容;

  3. 然后 C-c | 。



[5] C-y 或者选工具栏 Edit->Paste 。
