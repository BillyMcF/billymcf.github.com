简介

org-mode 主页上标题是 Org-Mode: Your Life in Plain Text 。

org-mode 是 Emacs 的一种模式(mode)，通过它我们可以很方便而且很高效的管理我们的笔记、ToDo列表，这对与我们要完成一个大工程很有帮助，因为我们可以通过计划表有序地进行。

我们还可以通过 org-mode 将我们写的笔记输出生成 html、pdf、DocBook 等格式的文件，所有的这些文件都来自同一个 .org 文件，而且我们不需要去编写这些文件相关的代码就可以完成。你正在看的这篇笔记就是通过 org-mode 编写和发布的，它自动生成了索引，而且我让它将索引编号，相信会对你的阅读提供方便。看，我一句代码都没有写就是这个效果，点击 这里 查看，这是不是很棒！

我从一开始使用 org-mode 就深深地被它吸引住了，这里我将我所知道的关于 org-mode 的用法记录下来，希望它能帮助你。当然我会不断地更新这里的内容，因为我发现 org-mode 实在有太多的东西可以去学习了。同时我也希望你将 org-mode 或者 emacs 一些可以提高我们工作效率的用法分享出来，因为我发现国内相关的介绍还是有点儿少。

安装

23版以后的 Emacs 已经开始默认包含有 org-mode 了，你只需要在你的配置文件 ~/.emacs 中加载它就行了。 

(require 'org-install)

或者你可以在 emacs 通过 M-x org-mode 加载 org-mode 。

现在重新打开你的 Emacs 新建一个以 .org 格式结尾的文件，在你的状态栏上应该会有诸如这样的 Org 字样显示 ,(其中的 yas 是 yasnippet 不全插件，我们暂时不用理它)这说明你的 Emacs 已经进入 org-mode 模式了。

使用

初体验

你可以在你的 Emacs 复制粘贴这个文件里的东西 点我 然后, M-x org-export-as-html(C-c C-e h)，然后在同目录下会生成一个 .html 文件，使用你的浏览器打开这个文件。怎么样，很简单吧！

标题

文档的标题可以这样设置 #+TITLE: 标题,更多的设置你可以使用快捷键输入 C-c C-e t 。

正如上面的示范文件一样，org-mode 里面的标题一首的一个 * 好，然后空格，输入的内容就是标题的东西了。一个星号是一级标题，两个是二级，一共十级。

列表

  • 无序列表以 '-'、 '+' 或者 '*' 开头
  • 有序列表以 '1.' 或者 1) 开头
  • 描述列表用 '::'
  • 同级列表缩进相同
  • 空两行后列表结束
  • 在同级里可以是用 M-RET (RET 表示 回车) 插入同级项

示例： 

** Lord of the Rings
   My favorite scenes are (in this order)
   1. The attack of the Rohirrim
   2. Eowyn's fight with the witch king
      + this was already my favorite scene in the book
      + I really like Miranda Otto.
   3. Peter Jackson being shot by Legolas
      - on DVD only
      He makes a really funny face when it happens.
   But in the end, no individual scenes matter but the film as a whole.
   Important actors in this film are:
   - Elijah Wood :: He plays Frodo
   - Sean Austin :: He plays Sam, Frodo's friend.  I still remember
     him very well from his role as Mikey Walsh in The Goonies.

列表/标题快捷键

列表/标题操作的一些快捷键
<TAB>折叠或者展开列表,如是在一空的项使用，则会变化各项的级别
M-<TAB>在当前级别里输入一个相同的项，如果在一项中间使用，该项将被分成两项
M-S-<RET>输入一项带 复选框 的项
M-<left>将该项提升一级
M-<right>将该项降低一级
M-S-<left>将该项提升一级，作用于下级所有项
M-S-<right>将该项降低一级，作用于下级所有项
M-S-<up>/<down>将该项与上/下一项交换位置

当光标处于列表项上时，使用以下快捷键，该级项及其所有下级项将被操作：

C-c C-c如果有复选框，触发
C-c -循环变换列表格式 ‘-’, ‘+’, ‘*’, ‘1.’, ‘1)’
C-c *将该列表项成标题，该标题成为所在标题的副标题
C-c C-*将整个列表变成子树
S-<left>/<right>将该项及其子项降低或提升一级
C-c ^对列表进行排序

标签

org-mode 里面的标签有以下几种： 

s    #+begin_src ... #+end_src 
e    #+begin_example ... #+end_example
q    #+begin_quote ... #+end_quote 
v    #+begin_verse ... #+end_verse 
c    #+begin_center ... #+end_center 
l    #+begin_latex ... #+end_latex 
L    #+latex: 
h    #+begin_html ... #+end_html 
H    #+html: 
a    #+begin_ascii ... #+end_ascii 
A    #+ascii: 
i    #+index: line 
I    #+include: line

是的，输入一个 <e 然后按 TAB 键就可以补全了，把 e 换成上面的每个字母都是一样的效果。很方便吧。

代码

org-mode 里面有代码的高亮显示，支持诸如 c, lisp, elisp, html等语言，更多支持的语言请看这个表： Languages

排版源码的标签及格式如下 

#+begin_src 语言
 ...
#+end_src

比如这样： 

#+begin_src c -n
int main(void)
{
  printf("Hello, World!\n");

  return 0;
}
#+end_src

出来的效果是这样的： 

1:  int main(void)
2:  {
3:    printf("Hello, World!\n");
4:  
5:    return 0;
6:  }

其中的 -n 是让源码显示行号。

如果是这样： 

#+begin_src c -t -h 7 -w 40
int main(void)
{
  printf("Hello, World!\n");

  return 0;
}
#+end_src

则会这样显示：

其中的 -t 是清除格式 -h 7 设置高度为 7 行，-w 40 设置宽度为 40 列。

表格

org-mode 里制作表格非常的方便和有趣，还可以自动计算total等，还可以自动编号。

建立表格

建立表格非常简单，表格的列以 | 隔开，按 TAB 跳到下一单元格， S-TAB 跳到上一单元格。新行 |- 然后 C-TAB 插入水平线，建立下面： 

| 操作             | 说明                     |
|------------------+--------------------------|
| M-S-<right>      | 插入一列                 |
| M-S-<left>       | 删除一列                 |
| M-<right>/<left> | 左右两列调换位置         |
| 行首插入         | - 然后 C-TAB  插入水平线 |

效果是这样的：

操作说明
M-S-<right>插入一列
M-S-<left>删除一列
M-<right>/<left>左右两列调换位置
行首插入- 然后 C-TAB 插入水平线

这里是表格更多的快捷键

格式输出

HTML

输出为 HTML 文件的快捷键是 C-c C-e h

使用 C-c C-e t 可以插入选项，其中可能有些选项我们需要经常用到：

H:标题层数
num:章节(标题)是否自动编号
toc:是否生成索引
creator:是否生成 "creat by emacs…"
LINK_UP:UP: 链接
LINK_HOME:HOME: 链接

更多的说明可以查看： orgmode.org

  • 有时我们需要包含 html 文件，可以使用 include 
#+include "~/wlatex/org-footer.html" html

记得在 后面加上 html

  • 如果需要为某个表格或者图标添加 class 可以这样设置 
#ATTR_HTML: class="center"

这样，生成的表格(或者其他)样式里面将有 center ， 你可以通过 css 文件控制。

  • 你也可以在你的 org 文件头部加入： 
#+STYLE: <style>table.center {margin-left:auto; margin- right:auto;}</style>
  • 当然，你可以包含某个 css 文件 
#+STYLE: <link rel="stylesheet" href="./css/style.css" type="text/css" />

几种字体格式 

/italy/ *bold* _underline_ =code= ~verbatim~ +strike-through+

效果 ： italy bold underline code verbatim strike-through

水平线 以5个以上连续的 - 组成

脚注

C-c C-x f 插入脚注。
C-c C-c 跳到脚注处。

PDF

发布成 pdf 的快捷键为 C-c C-e p ，在对应的目录下会生成 tex 及 pdf 文件。

可以自己定义 latex 样式，比如我这样定义： 

(add-to-list 'org-export-latex-classes
             '("bmc"
               "\\documentclass[12pt,a4paper,titlepage]{article}
\\usepackage{amssymb}
\\usepackage{amsmath}
\\usepackage{xeCJK}
\\usepackage{times}
\\usepackage{titlesec}
\\usepackage{titletoc}
\\usepackage{tocloft}
\\usepackage{fontspec}
\\usepackage{geometry}
\\usepackage{fancyhdr}
\\usepackage{indentfirst}
\\usepackage{listings}
\\usepackage{longtable}
\\usepackage[all]{draftcopy}
\\usepackage{xcolor}
\\usepackage{graphicx}
\\usepackage{fancyhdr}
\\usepackage[unicode=true,colorlinks=no,pdfborder=no,xetex,CJKbookmarks=true]{hyperref}

\\usepackage{fontspec,xunicode,xltxtra}
\\geometry{left=2.5cm,right=2.5cm,top=2.5cm,bottom=2.5cm} % 页面边距
\\XeTeXlinebreakskip = 0pt plus 1pt minus 0.1pt

\\setlength{\\parindent}{24pt}

\\lstset{
  % 行号
  numbers=left,
  numberstyle=\\tiny,
  % 背景框
  frame=shadowbox,
  rulesepcolor=\\color{red!20!green!20!blue!20},
  % 背景色
   backgroundcolor=\\color[RGB]{245,245,244},
  % 样式
  keywordstyle=\\bf\\color{blue},
  identifierstyle=\\bf,
  numberstyle=\\color[RGB]{0,192,192},
  commentstyle=\\color{blue!80},
  stringstyle=\\rmfamily\\slshape\\color[RGB]{128,0,0},
  % 显示空格
  showstringspaces=true,
  % escapeinside=``,
  extendedchars=false,
  basicstyle=\\scriptsize,
  % 边距
  xleftmargin=2em,xrightmargin=2em,aboveskip=1em,
  % 换行
  breaklines
}

% 一级章节设置
\\titleformat{\\section}{\\centering\\large\\bfseries\\setCJKmainfont{SimHei}}{第\\,\\thesection\\,部分}{12pt}{}
% 二级
\\titleformat{\\subsection}{\\small\\setCJKmainfont{SimHei}}{\\thesubsection}{10pt}{}
% 三级
\\titleformat{\\subsubsection}{\\small\\setCJKmainfont{SimHei}}{\\thesubsubsection}{9pt}{}
%%% 字体设置
\\setmainfont{Times New Roman}
\\setCJKmainfont{SimSun}
%%\\setCJKmonofont{DejaVu Sans YuanTi Mono}

\\renewcommand{\\contentsname}{目\\quad 录}   %将Contents换成 ”目录“
"
("\\section{%s}" . "\\section*{%s}")
("\\subsection{%s}" . "\\subsection*{%s}")
("\\subsubsection{%s}" . "\\subsubsection*{%s}")
("\\paragraph{%s}" . "\\paragraph*{%s}")
("\\subparagraph{%s}" . "\\subparagraph*{%s}")))

在你的 org 文件中，就可以在头部添加 #+LaTeX_CLASS: bmc 来引用这个样式。

(setq org-latex-to-pdf-process
'("xelatex -interaction nonstopmode %b"
"xelatex -interaction nonstopmode %b"))
