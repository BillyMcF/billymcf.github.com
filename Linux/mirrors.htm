Linux 镜像服务器

Author: Billy   Date: 2011-12-05

这里给出一些曾经用过的比较快的服务器，我的是教育网。

  • 目前正在用的(archlinux下): 
Server = http://ftp.jaist.ac.jp/pub/Linux/ArchLinux/$repo/os/$arch

这个好像是日本的一个镜像服务器，速度非常的快，最慢的时候也有 2M/s 以上的速度，最快的时候可以达到 10M/s 以上…以前使用 Gentoo 也是用的这个服务器，地震后慢了一阵，现在又恢复了。

  • 以前用过的:
    • 厦门大学信息与网络中心开源软件镜像服务 http://mirrors.xmu.edu.cn/

      这个的速度也非常快，有时也有几兆每秒，但是，没有 arch 的…

    • 北京交通大学镜像服务器 http://mirror.bjtu.edu.cn/

      新版的 arch 安装镜像里默认的中国地区镜像就是北交的，速度 200+ k/s，但十分稳定。

      以上是我在教育网环境下使用的镜像，仅供参考。
